# Tetromine's Tweaks
Vintage Story mod that patches both vanilla and some mods for better balance.

## Vanilla

### Hunter
Nerfed **Claustrophobic** (-15% ore drop rate, -10% mining speed) -> (-25% ore drop rate, -20% mining speed)


## Age of Flax Fork
