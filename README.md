# Tetromine's Tweaks
Vintage Story mod that patches both vanilla and some mods for better balance.

## Vanilla

### Hunter
- Nerfed **Claustrophobic** (-15% ore drop rate, -10% mining speed) -> (-25% ore drop rate, -20% mining speed)
- Replaced **Fleetfooted** (+10% movespeed) with **Furtive** (-35% animal detection range)

### Malefactor
- Buffed **Forager** (+10% loot from foraging, +20% wild crop drop rate) -> (+15% loot from foraging, +25% wild crop drop rate)
- Replaced **Furtive** (-35% animal detection range) with **Fleetfooted** (+10% movespeed)
- Added **Revolted** (-20% animal loot, -25% animal harvesting speed)

### Clockmaker
- Added **Industrialist** (-15% loot from foraging, -20% wild crop drop rate)

### Blackguard
- Buffed **Hardy** (+10% mining speed) -> (+20% mining speed)

### Tailor
- Removed **Civil** (-10% loot from foraging)
- Replaced **Kind** (-10% animal loot, -25% harvesting speed) with **Revolted** (-20% animal loot, -25% animal harvesting speed)


## XSkills
- Mixed Clay - Digger exclusive
- Gourmet - Chef exclusive
- Changed the descriptions of Finishing Touch and Fast Potter to make it more clear how they work

## Immersive Fibercraft
Changes handbook entries for loom and spinning wheel include all methods of spinning twine and creating linen.

## Knitting
Tiny tweak to lang files.

## Age of Flax Fork
Patches [Age of Flax Fork](https://mods.vintagestory.at/ageofflaxfork) to make the values more balanced compared to vanilla.
